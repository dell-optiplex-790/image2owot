Image2OWOT

What is this? ============================================================
A modified version of Image2ASCII, which pastes images into OWOT
(Our World Of Text).
==========================================================================

How do I build this? =====================================================
1. Download this repository
2. Open a terminal in its directory
3. Type in `npm i` to install dependencies, and then `npx webpack` in
   order to actually build.
==========================================================================

How do I bookmark this? ==================================================
1. create a bookmark
2. type in "javascript:"
3. open "dist/image2owot.bookmark.js" in a text editor
4. copy the content of the entire file into the URL box
   (keep "javascript:" as a prefix)
5. finish creating your bookmark
6. you're done!
==========================================================================
